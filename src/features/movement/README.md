# Movement Feature

Configurable movement physics parameters.

## CVars

- `_sf_sv_friction` - Player friction
- `_sf_sv_accel` - Player acceleration
- `_sf_sv_airaccel` - Air acceleration
- `_sf_sv_crouchspeed` - Crouch speed
- `_sf_sv_runspeed` - Run speed
- `_sf_sv_rampslide_threshold` - Ramp slide threshold
